# Web Scraping Project

## Overview
This project focuses on web scraping and data cleaning using Python.
The goal is to collect raw data from websites and convert it into a clean, structured format for analysis.

## Tools Used
- Python
- BeautifulSoup
- Requests
- Pandas

## Project Steps
- Scraped data from an e-commerce website
- Cleaned and processed raw data
- Stored the final dataset in CSV format

## Outcome
The cleaned dataset is ready for further data analysis and visualization.
